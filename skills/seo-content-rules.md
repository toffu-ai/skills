Apply these rules whenever writing or reviewing content intended to rank in organic search.

## Keyword Rules

- Every piece of content targets one primary keyword and 5-10 secondary keywords
- Include the primary keyword in the title, H1, first paragraph, and URL slug
- Use secondary keywords naturally throughout - never stuff them
- Match the content format to the search intent: informational queries get guides, transactional queries get product/comparison pages

## Structure Rules

- Title tag: under 60 characters, keyword near the front
- Meta description: 150-160 characters, includes the keyword and a reason to click
- URL slug: lowercase, hyphens, under 5 words, keyword-rich
- Use H2 and H3 headers to break up content. Every 200-300 words should have a header.
- Include an FAQ section for "People Also Ask" coverage when relevant
- Minimum 3 internal links per article, linking to related content on the site
- Add external links to authoritative sources where it supports claims

## Quality Rules

- Write for the reader first, search engines second
- Every section should answer a specific question or cover a distinct subtopic
- Include concrete examples, data, or screenshots where possible
- No filler paragraphs. If a section doesn't add value, cut it.
- Avoid thin content. Long-form pieces should be 1,500+ words. Short-form should still be substantive.

## Technical

- Images need alt text that describes the image (not keyword-stuffed)
- Use schema markup for FAQ, how-to, and review content when applicable
- Page should load in under 3 seconds. Flag heavy images or scripts.
